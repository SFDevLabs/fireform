Getting Started

The easiest way to get started is to drop the following code into your webpage. 

 
      <script type="application/javascript" src="https://src.fireform.org/fireform.js"></script>
      <script type="application/javascript">
      		new Fireform('#MySelector', 'http://fireform/formRepositoryLink');
      </script>

Change the DOM selector to point to your <form> tag and the formRepositoryLink to a fireform link.

Fireform will listen for a click event on the form's type="submit" input. Once submitted; the form's content will be uploaded, the submit button will be disabled and the success class will be added to the form.

You can submit the form with javascript too:

 
      <script type="application/javascript">
      		var fireform = new Fireform('#MySelector',  'http://fireform/formRepositoryLink');
      		fireform.submit();
      </script>

Full Working Example


  <!doctype html>
    <body>
      <form id="MySelector">
        <input type="text" name="givenname" placeholder="Name">
        <input type="text" name="message" placeholder="Message">
        <input type="submit">
      </form>

      <script type="application/javascript" src="https://src.fireform.org/fireform.js"></script>
      <script type="application/javascript">
          /* See more options at http://fireform.org/docs */
           var options = {
              simpleValidation:false,
              formValidationClass:"custom-submit-success",
              callback:function(err, val) { console.log(err,"callback") }
            }
          new Fireform('#MySelector', 'http://fireform.org/publicexample', options)
      </script>
  </body>

Options

You can pass an options object into the Fireform function. 

 
      <script type="application/javascript">
      		options={
      			//String: Class applied to form after submission.

      			successClass:'success-class', 
			failedClass:'failed-class',  //String: Class applied to form upon failer.
			formValidationClass:'validation-class', //String: Class applied to form on bad validation.
			inputValidationClass:'validation-class', //String: Class applied to input on bad validation.
			simpleValidation:true //Bool: Simple validation to ensure all inputs have a value.
			//Send an email to this address when a new form is posted.*
			emailNotification: 'myemail@example.com'
			//Send an email confirmation to a specific email collected in the form.*
			emailConfirmationName:'emailInputName'//String: Name attribute of the email input.
			emailConfirmationSubject:'Subject Text'//String: Email subject.
			emailConfirmationBodyHTML:'Body HTML'//String: Shows in most modern email clients.
			emailConfirmationBodyText:'Body Text'//String: Body Text fallback.
			callback: function(err, val){ } //Function: Fired after successful submission.
      		}

      		new Fireform('#MySelector', 'http://fireform/formRepositoryLink', options);
      </script>

*Email Notifications and Confirmations must be activated on the form's table page.
